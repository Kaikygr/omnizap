# async deleteChatData(chatJid) → {Promise.<void>}

Remove todos os dados associados a um chat específico do banco de dados. A remoção é em cascata devido às chaves estrangeiras ON DELETE CASCADE:

  • Se o chatJid pertencer a um grupo (termina com '@g.us'), a entrada correspondente na tabela Groups (e, por cascata, GroupParticipants) é removida.
  • A entrada na tabela Chats é removida.
  • Por cascata, todas as mensagens (Messages) e recibos de mensagens (MessageReceipts) associados a este chatJid também são removidos.

# async initialize() → {Promise.<void>}

  1. Cria uma conexão temporária com o servidor MySQL para verificar se o banco de dados especificado em this.dbName existe. Se não existir, o banco de dados é criado.
